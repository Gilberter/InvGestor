### Ejemplo: `set` y `list` (Python)

Explicación rápida:

* **Listas (`list`)**: colecciones ordenadas y mutables que pueden contener duplicados.
* **Conjuntos (`set`)**: colecciones no ordenadas y sin duplicados; operaciones en conjuntos (unión, intersección) son eficientes.

Ejemplo de código:

```python
# Lista (list)
frutas = ["manzana", "pera", "naranja", "manzana"]
print("Lista:", frutas)  # conserva duplicados y orden

# Conjunto (set)
conjunto_frutas = set(frutas)
print("Set:", conjunto_frutas)  # elimina duplicados, no garantiza orden

# Operaciones con sets
a = {1, 2, 3}
b = {3, 4, 5}
print("Unión:", a | b)
print("Intersección:", a & b)
print("Diferencia (a-b):", a - b)
```
